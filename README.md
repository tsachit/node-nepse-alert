# Node NEPSE Alert

Just an alert system checking using NEPSE data

## How to run

Run these commands

one time command
```
yarn install
```
and to run
```
yarn start
```

Note: Make changes in `index.js` to change the `interval` value. If you want to add/update/remove conditions change `conditions.json` file.
